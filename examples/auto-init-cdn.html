Widgets automatically initialize using data attributes

How it works: Add data-vapor-widget attribute to any div, along with data-api-key and other configuration attributes. The widget will automatically initialize when the page loads!

Default Theme (Auto)

<div data-vapor-widget data-api-key="vf_test_xxx" data-network="sepolia" data-theme="auto" ></div>

Light Theme

<div data-vapor-widget data-api-key="vf_test_xxx" data-theme="light" ></div>

Dark Theme

<div data-vapor-widget data-api-key="vf_test_xxx" data-theme="dark" ></div>

Compact Mode

<div data-vapor-widget data-api-key="vf_test_xxx" data-compact="true" ></div>
